# 🌐 Network Engineering & Infrastructure Portfolio

Welcome to my networking portfolio! I am a Computer Science student at Universiti Teknologi Malaysia (UTM) with a strong foundation in routing, switching, and cybersecurity concepts through the Cisco Networking Academy. 

This repository serves as a documented timeline of my hands-on network engineering labs, architecture designs, and protocol troubleshooting using Cisco Packet Tracer and enterprise hardware concepts.

## 🛠️ Core Competencies
* **Layer 2 Switching:** VLAN Segmentation, 802.1Q Trunking, VTP, Spanning Tree Protocol (STP), EtherChannel (LACP/PAgP).
* **Layer 3 Routing:** Inter-VLAN Routing (Router-on-a-Stick), Static Routing, Default Gateways, Subnetting.
* **Network Security:** Native VLAN Hardening, Access Port Lockdown, Management Interface (SVI) isolation.
* **Tools:** Cisco IOS CLI, Cisco Packet Tracer.
